Understanding the <pre> Element

The <pre> Element Explained

The <pre> element in HTML is used to display text in a pre-formatted way. This means that whitespace (spaces, tabs, newlines) and line breaks within the <pre> tags are preserved and rendered exactly as they appear in the HTML source code. It's often used for displaying code snippets, ASCII art, or any text where the formatting is important.ASCII stands for American Standard Code for Information Interchange. Number are only 0 to 31.

Code Formatting Comparison

Here's a snippet of JavaScript code displayed using a regular paragraph tag:

function greet (name) { console.log ('Hello, ${name!}')}

Now, here's the same code within a <pre> tag:

function greet(name) {
  console.log('Hello, ${name}');
}
